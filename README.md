# Character Roleplay

A web application for character roleplay interactions. Visit the live site at: https://what11233.github.io/redesigned-succotash

## About

This is a full-stack web application featuring a React frontend and Express backend with TypeScript support.

## Deployment

This repository is configured for GitHub Pages. The static files are automatically deployed to `https://what11233.github.io/redesigned-succotash`
